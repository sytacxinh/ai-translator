Tôi có nâng cấp tiếp theo này muốn làm.

```
1.History/Favorites: Lưu lại các bản dịch đã thực hiện, chỉ lưu 100 bản dịch gần nhất để chống tràn bộ nhớ. Tạo 1 nút gọi các bản dịch gần nhất, scroll xuống dưới để xem phần phía dưới. nút này nằm trong window dịch chính.
2.Dictionary mode: nếu phần dịch chỉ có một đến vài từ rất ít, thì sửa promt sao cho thành Dịch từ đơn có thêm giải nghĩa, ví dụ, phát âm, mang tính từ điển cho user dễ hiểu
Tạo thêm 2 row Primary đặc biệt, 1 là xử lý hình ảnh, 1 là xử lý file. Phía dưới mỗi Primary mới này có nút add xử lý ảnh và file backup.
3.Tạo 1 toggle cho phép bật chế độ nâng cao trong tab API. Toggle này chỉ có thể bật khi user nhập API có khả năng xử lý hình ảnh và test thành công. Sau khi bật, Bên trong window dịch chính sẽ hiện thêm phần add hình ảnh. 
3.1. OCR: Chụp màn hình → dịch text trong ảnh (dùng Tesseract hoặc AI vision) (tôi chưa rõ cách thức tốt nhất, hãy đề xuất). ví dụ sử dụng model AI nào tốt nhất, hình như các model ai có các model dịch ảnh thì phải...
4.Tạo 1 toggle cho phép bật chế độ nâng cao trong tab API. Toggle này chỉ có thể bật khi user nhập API có khả năng xử lý files. Sau khi bật, Bên trong window dịch chính sẽ hiện thêm phần add hình ảnh. 
(Chỗ logic này check lại giúp tôi, một API xử lý hình ảnh có phải luôn đồng thời xử lý file luôn không? hay tôi đã làm đúng rồi?)
4.1.Batch translate: Dịch nhiều file cùng lúc (txt, docx, srt subtitles) Tạo 1 nút cho phép nhập browse file ở trong cửa sổ dịch chính, chỉ cần kéo file vào thả cũng được.
Khu vực chứa file và ảnh nằm cùng 1 chỗ và hiện thị bản mini giống như khi sử dụng chatGPT vậy.Hiện thị phía trên phần Original textbox.

```
