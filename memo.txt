AI Translator v1.7.0 - Project Structure
========================================

gemini_translator_v4/
├── main.py                  # Entry point
├── translator.py            # Backward compatible
├── config.py                # Configuration management
├── requirements.txt         # Dependencies
│
└── src/
    ├── constants.py         # VERSION, LANGUAGES, PROVIDERS_LIST, etc.

    │
    ├── core/
    │   ├── clipboard.py     # ClipboardManager

    │   ├── hotkey.py        # HotkeyManager (global hotkeys)
    │   ├── history.py       # Translation history (100 entries)
    │   ├── multimodal.py    # Vision processing (OCR)

    │   ├── file_processor.py # Document text extraction
    │   ├── pdf_ocr.py       # [NEW] Scanned PDF OCR support
    │   ├── crypto.py        # Secure API key storage (DPAPI)
    │   ├── ssl_pinning.py   # SSL certificate pinning
    │   ├── auth.py          # [NEW] Windows Hello authentication
    │   └── provider_health.py # [NEW] Smart provider fallback
    │
    ├── ui/
    │   ├── settings.py      # SettingsWindow (with Show All API Keys)
    │   ├── dialogs.py       # APIErrorDialog
    │   ├── attachments.py   # File attachment widget
    │   ├── history_dialog.py # History viewer (with search)
    │   ├── progress_dialog.py # Progress indicator
    │   └── toast.py         # [NEW] Toast notifications
    │
    └── utils/
        ├── logging_setup.py # setup_logging()
        ├── single_instance.py # is_already_running()
        └── updates.py       # check_for_updates(), download_and_install_update()

========================================
New Features in v1.7.0
========================================

1. Windows Hello Authentication
   - PIN, Fingerprint, Face recognition
   - Protects "Show All API Keys" and "Delete All API Keys" buttons
   - Falls back to Windows password if Windows Hello unavailable
   - Soft rate limit: 5 attempts per dialog

2. Show All API Keys Button
   - One-click to show/hide all API keys
   - Syncs with individual Show/Hide buttons
   - Requires authentication only when showing

3. Smart Provider Fallback
   - Auto-switch to backup API when primary fails
   - Provider health tracking
   - Configurable retry logic

4. Toast Notifications
   - Non-intrusive notifications
   - Auto-dismiss with timer
   - Click to dismiss

5. Scanned PDF OCR
   - Extract text from scanned/image-based PDFs
   - Uses AI vision for text recognition
   - Falls back to standard PDF text extraction

6. Search History
   - Filter translation history by keyword
   - Search in both original and translated text
